* Install & Recovery
** ISO Installation
|---------+------------------------|
| lsblk   | list block devices     |
| mkfs    | format partition       |
| fdisk   | manage partition table |
|---------+------------------------|

#+begin_src sh
  # Inspect
  lsblk
  sudo fdisk -l

  # Format whole disk or partition
  sudo mkfs -t ext4 /dev/sda
  sudo mkfs -t ext4 /dev/sda1

  # Write ISO to USB
  sudo dd if=ubuntu.iso of=/dev/sda bs=4M status=progress && sync

  # Partition with fdisk (GPT)
  sudo fdisk /dev/sda   # then: m → g → n → 1 → Enter → w

  # Clone a partition
  sudo dd if=/dev/nvme0n1p1 of=/dev/sda1 bs=4M status=progress
#+end_src

From Windows: use Rufus.
To wipe a disk: =diskpart= → select disk → clean.

** Disk Backup & Restore
Backup system partition to USB:
#+begin_src sh
  sudo dd if=/dev/nvme0n1p2 of=/dev/sda2 bs=4M status=progress && sync
#+end_src

Full workflow:
1. =lsblk= / =sudo fdisk -l= — inspect disks
2. =sudo mkfs -t ext4 /dev/sda= — format target disk
3. =sudo fdisk /dev/sda= — create partitions on target
4. =sudo dd if=/dev/nvme0n1p2 of=/dev/sda2 bs=4M status=progress && sync= — backup
5. Use Disks (GUI) to resize the restore partition (~10 GB larger than the backup)
6. To restore: =sudo mkfs -t ext4 /dev/nvme0n1p2= then reverse the =dd= command

Alternative with rsync:
#+begin_src sh
  sudo mount /dev/sda1 /media/Backup
  rsync -aAXv --delete --exclude=/media / /media/Backup/LinuxBackup
#+end_src

* Development
** GCC Basics
#+begin_src sh
  gcc -c file.c -o file.o          # compile only
  gcc file1.o file2.o -o app       # link
  gcc -I ./include -L ./lib -lmylib main.c -o app   # with headers and library
#+end_src

** Static Libraries (.a)
1. Compile sources to object files: =gcc *.c -c -I ../include=
2. Archive: =ar rcs libtest.a *.o=
3. Verify: =nm libtest.a=
4. Consumer compiles: =gcc main.c -I ./include/ -L ./lib/ -ltest -o app=
   - =-ltest= strips =lib= prefix and =.a= suffix

*Pros:* fast load, easy to distribute. *Cons:* duplicates disk usage, update needs re-link.

** Dynamic / Shared Libraries (.so)
Build:
#+begin_src sh
  gcc -c -fPIC *.c -I ../include        # position-independent code
  gcc -shared *.o -o libxxx.so
#+end_src

Consumer compiles:
#+begin_src sh
  gcc -I ./include -L ./lib -lxxx main.c -o app
#+end_src

Runtime — the loader must find the =.so=:
- Per-session: =export LD_LIBRARY_PATH=./lib=
- Permanent (user): add to =~/.bashrc=
- Permanent (system): add to =/etc/ld.so.conf=, run =sudo ldconfig -v=
- At link time: =-Wl,-rpath=/absolute/path/to/lib=

One-shot build example:
#+begin_src sh
  gcc -c -fPIC add.c sub.c mult.c divi.c
  gcc -shared -o libmymath.so add.o sub.o mult.o divi.o
  gcc -L. -Wl,-rpath=. -Wall -o mathDemo mathDemo.c -lmymath
  ./mathDemo
#+end_src

*Note for Go:* CGo only supports =LD_LIBRARY_PATH=; =-Wl,-rpath= doesn't carry through.

** dlopen / dlsym (runtime loading)
Functions: =dlopen=, =dlclose=, =dlsym= — load a shared library at runtime and call its symbols.

** Makefile
Format:
#+begin_src makefile
  target: dependencies
  	command
#+end_src

V1 — simplest:
#+begin_src makefile
  app: main.c add.c sub.c mul.c
  	gcc main.c add.c sub.c mul.c -o app
#+end_src

V2 — incremental (only recompile changed files):
#+begin_src makefile
  app: main.o add.o sub.o mul.o
  	gcc main.o add.o sub.o mul.o -o app
  %.o: %.c
  	gcc -c $< -o $@
#+end_src

V3 — automatic variables and wildcard:
#+begin_src makefile
  src = $(wildcard ./*.c)
  obj = $(patsubst %.c, %.o, $(src))
  target = app

  $(target): $(obj)
  	gcc $^ -o $@
  %.o: %.c
  	gcc -c $< -o $@

  clean:
  	-rm $(obj) $(target) -f
#+end_src

| variable | meaning           |
|----------+-------------------|
| $@       | target            |
| $<       | first dependency  |
| $^       | all dependencies  |

* IPC & Concurrency
** Concepts
- *Parallel:* multiple processes running simultaneously (multi-core)
- *Concurrent:* multiple threads interleaved on one core
- *Synchronous / Asynchronous:* whether a single thread waits for I/O

** Multi-Process Communication
*** Message Queue (SysV)
Key API:
| function  | purpose                      |
|-----------+------------------------------|
| ftok()    | generate IPC key             |
| msgget()  | create/open queue            |
| msgsnd()  | send message (by type)       |
| msgrcv()  | receive message (by type)    |
| msgctl()  | destroy queue                |

Message struct must start with =long type=.

One-way: server sends type 100, client receives type 100.
Two-way: fork server → parent sends type 100, child receives type 200; client does the opposite.

*** Named Pipe (FIFO)
#+begin_src sh
  mkfifo ./myfifo
#+end_src
One process opens =O_WRONLY=, another =O_RDONLY=.

*** Unnamed Pipe
#+begin_src C
  int fd[2];
  pipe(fd);
  // fd[0] for read, fd[1] for write
  // Write first to fd[1], then read from fd[0]
#+end_src

** Multi-Thread (pthreads)
Compile with =-pthread=.

| function                       | purpose                        |
|--------------------------------+--------------------------------|
| pthread_create()               | spawn thread                   |
| pthread_join()                 | wait and retrieve return value |
| pthread_detach()               | detach (cannot join)           |
| pthread_exit()                 | exit thread                    |
| pthread_cancel()               | request cancellation           |
| pthread_setcancelstate()       | enable/disable cancellation    |
| pthread_setcanceltype()        | deferred vs asynchronous       |
| pthread_self()                 | get own thread ID              |

Mutex:
#+begin_src C
  pthread_mutex_t mutex = PTHREAD_MUTEX_INITIALIZER;
  pthread_mutex_lock(&mutex);
  // critical section
  pthread_mutex_unlock(&mutex);
#+end_src

Read-Write Lock:
#+begin_src C
  pthread_rwlock_t rwlock;
  pthread_rwlock_init(&rwlock, NULL);
  pthread_rwlock_wrlock(&rwlock);   // exclusive
  pthread_rwlock_rdlock(&rwlock);   // shared
  pthread_rwlock_unlock(&rwlock);
  pthread_rwlock_destroy(&rwlock);
#+end_src

Thread attributes:
#+begin_src C
  pthread_attr_t attr;
  pthread_attr_init(&attr);
  pthread_attr_setdetachstate(&attr, PTHREAD_CREATE_JOINABLE); // or DETACHED
  pthread_attr_setstacksize(&attr, 100000);                    // if _POSIX_THREAD_ATTR_STACKSIZE
  pthread_create(&tid, &attr, func, arg);
  pthread_attr_destroy(&attr);
#+end_src

** UWSGI + Nginx + Django
References:
- https://tonyteaches.tech/django-nginx-uwsgi-tutorial/
- https://www.digitalocean.com/community/tutorials/how-to-secure-nginx-with-let-s-encrypt-on-ubuntu-20-04

*Setup:*
1. Ensure =python manage.py runserver= works
2. Install uwsgi (not in venv): =sudo pip install uwsgi=
3. Test: =uwsgi --http :8000 --module project.wsgi=
4. Config at =/etc/uwsgi/apps-available/project.ini=:
   #+begin_src ini
     [uwsgi]
     chdir = /home/user/project
     module = project.wsgi
     home = /home/user/venv
     master = true
     processes = 10
     socket = /home/user/project/project.sock
     chmod-socket = 666
     vacuum = true
     logto = /var/log/uwsgi/project.log
   #+end_src
5. Nginx config:
   #+begin_src nginx
     upstream django {
         server unix:///home/user/project/project.sock;
     }
     server {
         listen 80;
         server_name example.com;
         location /static { alias /home/user/project/static; }
         location /media  { alias /home/user/project/media; }
         location / {
             uwsgi_pass django;
             include /home/user/project/uwsgi_params;
         }
     }
   #+end_src
6. Systemd service at =/etc/systemd/system/uwsgi.service=:
   #+begin_src ini
     [Unit]
     Description=uWSGI Emperor service
     [Service]
     ExecStartPre=/bin/bash -c 'mkdir -p /run/uwsgi; chown user:user /run/uwsgi'
     ExecStart=/usr/local/bin/uwsgi --emperor /etc/uwsgi/apps-enabled
     Restart=always
     KillSignal=SIGQUIT
     Type=notify
     NotifyAccess=all
     [Install]
     WantedBy=multi-user.target
   #+end_src
7. Enable: =sudo systemctl start uwsgi && sudo systemctl enable uwsgi=

* Git
** Setup
#+begin_src sh
  sudo apt install git
  git config --global user.name "name"
  git config --global user.email "email"
  ssh-keygen -t rsa -C "email"           # add pubkey to GitHub
#+end_src

** Basic Workflow
#+begin_src sh
  git init
  git remote add origin git@github.com:user/repo.git
  git add -A
  git commit -m "message"
  git push -u origin master
#+end_src

Amend last commit (no new commit record):
#+begin_src sh
  git commit --amend
#+end_src

** Branching
#+begin_src sh
  git branch development
  git checkout development
  # ... work, commit, push ...
  git push -u origin development

  # Merge to master
  git checkout master
  git pull origin master
  git merge development
  git push origin master

  # Delete branch
  git branch -d development
  git push origin --delete development
#+end_src

Track remote branch:
#+begin_src sh
  git checkout -t origin/ui-mockup
#+end_src

** Recovery
| state             | undo command                  |
|-------------------+-------------------------------|
| modified (unstaged)| git checkout -- file         |
| staged            | git reset HEAD file           |
| committed         | git reset --hard HEAD~1       |
| deleted (local)   | git reflog → git reset --hard <id> |
| remove from index | git rm --cached file          |

* Troubleshooting Collection
** System limit for number of file watchers reached
#+begin_src sh
  sudo sysctl -w fs.inotify.max_user_watches=100000
#+end_src

** Right-click on touchpad not working
#+begin_src sh
  gsettings set org.gnome.desktop.peripherals.touchpad click-method areas
#+end_src

** Package manager lock / broken packages
#+begin_src sh
  sudo apt --fix-broken install
#+end_src
